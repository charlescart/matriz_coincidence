# Ejercicio
Dadas dos matrices ordenadas de enteros, verifique si hay al menos un elemento que ocurre en ambas matrices.

# Ejemplo

Para 
```
arr1 = [1, 2, 3, 5]
```
```
arr2 = [1, 4, 5]
```
La salida debe ser:
```
checkSameElementExistence(arr1, arr2) = true;
```
#
Para 
```
arr1 = [1, 3, 5]
```
```
arr2 = [-2, 0, 2, 4, 6]
```
La salida debe ser:
```
checkSameElementExistence(arr1, arr2) = false;
```
